<objective>
Fix two critical Windows production build bugs: (1) CMD windows flashing when executing internal commands, (2) installation verification hanging after package manager install.

Purpose: These bugs block the beta release — they make the app feel broken and unusable on Windows. In dev builds everything works fine because the app runs inside an existing console; in production builds, each `cmd /c` spawn creates a visible console window, and shell-wrapped commands can hang indefinitely without timeouts.

Output: All 12 Rust source files migrated to use `CREATE_NO_WINDOW` flag via a shared `silent_cmd()` helper, with 30s/120s timeouts on all command executions.
</objective>

  <name>Task 1: Create silent command helper module</name>
  <read_first>src-tauri/src/commands/mod.rs</read_first>
  <files>src-tauri/src/commands/silent.rs, src-tauri/src/commands/mod.rs</files>
  <action>
    Create `src-tauri/src/commands/silent.rs` with:

    1. `silent_cmd(program: &str) -> tokio::process::Command`:
       - On Windows (`#[cfg(target_os = "windows")]`): Creates `tokio::process::Command::new(program)` with `.creation_flags(0x08000000)` (CREATE_NO_WINDOW)
       - On non-Windows: Creates `tokio::process::Command::new(program)` with no flags
       - Both branches set `.stdout(Stdio::piped())` and `.stderr(Stdio::piped())` by default

    2. `run_with_timeout(cmd: tokio::process::Command, timeout_secs: u64) -> Result<Output, String>`:
       - Wraps `tokio::time::timeout(Duration::from_secs(timeout_secs), cmd.output())`
       - Returns descriptive error on timeout: "Command timed out after {N}s"
       - Maps inner errors to strings

    3. Constants: `QUICK_TIMEOUT: u64 = 30`, `INSTALL_TIMEOUT: u64 = 120`

    4. Add `use std::process::Stdio;` and `use tokio::time::{timeout, Duration};`

    Then add `pub mod silent;` to `src-tauri/src/commands/mod.rs` (alphabetically between `platform` and `system_check`).
  </action>
  <verify>
    <automated>cargo check --manifest-path src-tauri/Cargo.toml 2>&1 | head -30</automated>
  </verify>
  <done>
    Module compiles. `silent_cmd()` returns a Command with CREATE_NO_WINDOW on Windows. `run_with_timeout()` returns timeout errors for slow commands. `pub mod silent;` is in mod.rs.
  </done>
